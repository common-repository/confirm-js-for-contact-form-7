A Javascript plugin that provides a confirmation screen for the Contact Form 7 plugin. Display a confirmation screen on Contact Form 7 with reCAPTCHA (v3) enabled.

== Description ==

Add a confirmation screen that works with Javascript to the Contact Form 7 form.
Since it is different from the judgment method of Contact Form 7, an error may occur again when actually sending.
The confirmation screen can be operated even when reCAPTCHA (v3) is enabled.
The messages displayed at this time are available in English and Japanese.

Please note that the author of this plugin is not related to the developer of Contact Form 7 plugin.

= Privacy notices =

This plugin itself does not do the following.

* Tracks the user by stealth.
* Write the user's personal information to the database.
* Send data to external servers.
* Use cookies.


== Installation ==

1. Upload the entire `confirm-js-for-contact-form-7` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
